# 🌐 Personal Portfolio

A modern and responsive personal portfolio website built using HTML and CSS. This portfolio showcases my skills, projects, and contact information in a clean and professional design.

## 🚀 Features

- Responsive Navigation Bar
- Hero Section with Profile Image
- Skills Section
- Projects Showcase
- About Me Section
- Contact Form
- Footer with Quick Links
- Smooth Hover Effects
- Modern UI Design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Basic Knowledge)
- TypeScript
- Python
- Sanity CMS
- Streamlit
